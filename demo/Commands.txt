Create my file to push up
`touch <filename>`

Add program to staging area
`git add <filename>`
`git add .` Adds EVERYTHING HERE 

Check the status of our repo
`git status`

Commit my changes
`git commit -m "commit message"`

Push my changes up to github
`git push`
